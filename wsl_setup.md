# Install an Image

- To see what is installed locally

```bash
  wsl --list   
```

- To see what is available from Microsoft

  ```bash
   wsl --list --online
  ```

- To install an image pick an image that is not installed. 

```bash
wsl --install Ubuntu-24.04  
```
